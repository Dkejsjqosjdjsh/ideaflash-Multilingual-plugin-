## 您需要知道的一切

### 📦 這是什麼？

一個超大的 Sea Lantern 插件，包含 **233 種語言**，為 Minecraft 啟動器提供全球化支援。

### 🎯 3 步安裝

```
1️⃣  下載: sealantern-language-pack-expanded-1.0.0.zip (5 MB)
2️⃣  解壓: 到 ~/.sealantern/plugins/ 目錄
3️⃣  重啟: Sea Lantern 啟動器
```

### 🔧 安裝位置

找到您的 Sea Lantern 插件目錄：

**Windows**: 
```
C:\Users\[用戶名]\AppData\Roaming\Sea Lantern\plugins\
或
~/.sealantern/plugins/
```

**macOS**:
```
~/Library/Application Support/Sea Lantern/plugins/
```

**Linux**:
```
~/.config/sealantern/plugins/
```

### ✅ 安裝驗證

安裝完成後，打開 Sea Lantern：
1. 進入 ⚙️ 設置
2. 找到「Language」或「語言」
3. 應該看到 230+ 語言選項
4. 試試選擇任意語言（如「日本語」）
5. UI 應該立即更新

### ❓ 常見問題

**Q: 會影響 Sea Lantern 性能嗎？**
A: 不會。額外占用只有 10-15 MB 內存，不影響遊戲啟動。

**Q: 支援我的語言嗎？**
A: 很可能！233 種語言涵蓋世界 80%+ 人口。詳見完整列表。

**Q: 可以添加更多語言嗎？**
A: 可以！我們歡迎貢獻。詳見 CONTRIBUTING.md。

**Q: 可以離線使用嗎？**
A: 完全可以。所有內容都在本地，無需網絡。

**Q: 如何卸載？**
A: 只需刪除 `~/.sealantern/plugins/sealantern-language-pack-expanded/` 文件夾。
